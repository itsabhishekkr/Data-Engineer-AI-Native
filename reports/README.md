# AI-Native Data Quality System for News Events

## Overview

This project builds an AI-native data quality system for a news events dataset.

It combines:

- Rule-based checks
- LLM-based semantic checks
- Agentic remediation
- Eval harness
- Observability logs
- Monitoring
- Human review queue
- Dashboard
